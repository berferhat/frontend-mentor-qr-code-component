# QR Code Component

A simple QR code card component based on the [Frontend Mentor](https://www.frontendmentor.io) design. Centered white card with a QR code image and text.

**Live demo:** [https://frht-frontend-mentor-qr-code-component.netlify.app](https://frht-frontend-mentor-qr-code-component.netlify.app)

## Preview

![QR Code Component](./preview.jpg)

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS variables, aspect-ratio)
- [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts)

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/berferhat/frontend-mentor-qr-code-component.git
   cd frontend-mentor-qr-code-component
   ```
2. Open `index.html` in your browser or use a local server (e.g. Live Server).

## Features

- Responsive card (max-width: 320px)
- Light shadow and rounded corners
- Attribution at the bottom of the page
- Semantic HTML and accessible structure
